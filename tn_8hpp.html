Eigen type as \(T(n)\) Lie Group.

Note
Tn is a typedef for an Eigen column vector and therefore does not have the Lie group API from LieGroupBase. lie_traits<Tn<N, Scalar>> defines the operations corresponding to the Lie group \(T(n)\).

Memory layout

  • Group: \( \mathbf{t} = [x_1, \ldots, x_n] \)
  • Tangent: \( \mathbf{v} = [v_1, \ldots, v_n] \)

Lie group Matrix form

\[ \mathbf{X} = \begin{bmatrix} I & \mathbf{t} \\ 0 & 1 \end{bmatrix} \]

Lie algebra Matrix form

\[ \mathbf{v}^\wedge = \begin{bmatrix} 0 & \mathbf{v} \\ 0 & 0 \end{bmatrix} \]
